# Personal Assistant with Cortex Memory

A simple personal assistant that remembers facts, decisions, and preferences across sessions.

## Setup

1. Start Cortex: `cortex serve`
2. Install deps: `pip install -r requirements.txt`
3. Set your OpenAI key: `export OPENAI_API_KEY=sk-...`
4. Run: `python assistant.py`

## Features

- Remembers facts, decisions, and preferences you tell it
- Gets a briefing at the start of each session
- Stores important information automatically
- Knowledge persists between restarts
